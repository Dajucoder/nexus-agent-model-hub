# 许可证说明

## 默认许可证

本仓库默认采用 **PolyForm Noncommercial 1.0.0**。

官方来源：

- https://polyformproject.org/licenses/noncommercial/1.0.0/
- https://polyformproject.org/wp-content/uploads/2020/05/PolyForm-Noncommercial-1.0.0.txt

## 实务含义

无需额外商业授权即可进行的使用：

- 个人学习
- 研究与实验
- 非商业目的内部测试
- 教育、公益、政府、公共研究机构使用

默认许可证不允许的使用：

- 直接售卖本软件
- 作为付费或营收支持的托管服务提供
- 嵌入商业 SaaS 或商业软件产品
- 在营利性业务中为商业利益运行
- 面向付费客户提供托管运维或代运营

最后三条是基于官方 “noncommercial” 限制做出的操作性解释，用于帮助使用者理解边界，不构成法律意见。

## 为什么选择它

- 它符合“严格不可商用”的明确要求
- 它避免把 AGPL 误写成“禁止商用”这种常见错误
- 它对托管云服务和商业化使用的边界更直接

## 与常见许可证对比

- `MIT`、`Apache-2.0`：允许商业使用
- `AGPL-3.0`：要求网络服务修改版开源，但并不天然禁止商业使用
- `SSPL`：强调服务端源码开放义务，不等于普遍不可商用
- `PolyForm Noncommercial`：默认只允许非商业目的使用，对商业场景限制更强

## 如需商业使用

建议在仓库 README 中额外给出商业授权联系地址，并通过仓库外的商业许可协议授予权利。
